# PresenterLights
A traffic light system for presenters

Green - You're okay for time

Amber - You have 30 seconds left

Red - You're out of time


# Build
Requirements:
  1. NodeJS
  2. Angular CLI

## Server

```bash
$ cd server
$ npm install -g gulp-cli
$ npm install
$ gulp build
$ npm start
```

## Client

```bash
$ cd client
$ npm install
$ ng serve
```
